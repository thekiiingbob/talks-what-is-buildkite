# What's an example pipeline?

```yaml
steps:
  - label: ':hammer: Tests'
    command: 'scripts/tests.sh'

  - wait

  - label: ':package: Package'
    command: 'scripts/build-binaries.sh'

  - wait

  - block: ':shipit: Release'
    branches: 'master'

  - label: ':github: Release'
    command: 'scripts/build-github-release.sh'
    branches: 'master'
```
